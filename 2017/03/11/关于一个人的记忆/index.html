关于一个人的记忆

发表于 2017-03-11   |   分类于 随笔   |     |   阅读次数 
2017年2月23日21:55:00
歌名：房东的猫《秋酿》

那年我才十四，你也正值青春年少。

已经记不起第一次见到你，是什么时候，是什么地点。那一年关于你的记忆只有在教室里，我坐在教室的第一组第三排，就坐在第二组第三排，但是我在第一列，而你在第四列。记忆中你总喜欢扎着歪向一边的马尾，说话的时候总是带有一种一本正经的感觉，笑起来有一个停顿的间隙，简单可爱，做事总是很认真。那段认识你的时光里，算是我生命中比较快乐的日子，也许那段时光就叫青春吧。

那时，我第一次知道你的名字的时候，我第一个想到的成语，就是一见钟情，因为里面有属于你的姓。

你还记得我们那时早上有英语的背书吗？我一直都很懒，我一直都没有去背，也许是因为我这是我第二个初一的缘故，学习的东西都是我已经学过了，对于这种形式上的东西，我都不是很注意。但是有一天，我看到你很认真地在背书，我就到教室后面的那张表上，看看你的名字下的打钩√，你就是为数不多的追着老师进度背的那几个。后来，每天早上我都很认真地背书，那是因为你啊，但是你不知道。

你还记得你曾经在有一次体育课结束后对我说的话？那时我上体育课，喜欢穿得特别一点，就是我穿着一件T-shirt打底，然后外面套一件球服，就是背心。也许当时有一种标新立异的感觉，就是觉得这样回头率会高一点，我也忘了我穿了几次。那一次，体育课结束后，你应该是从小卖部出来，我刚刚走到教学楼的那个角，我们不期而遇。但是你身边还有其他女同学，你当时对我说，穿衣服穿得这么奇怪。这就是我想要的引起你的注意啊。后来，我都没有再这样穿过了，那是因为你啊，但是不知道。

初二的时候，我们不在同一个班了，有时我经过你班门口，你就在门口和别人聊天，我都很匆忙地走过，我有点怕你发现我的惊慌失措。

初三，我们在同一个班，但是我一点印象都没有，也许我都把心思放在学习上了。我记得有一次，化学老师做一个实验，具体是什么实验，我也忘了。老师当时叫了去点火，我在上面不知道是紧张，还是怎么的，点火的时候，碰到了那个倒扣的杯子，你当时在第一排，一脸怨气地说我，都是你不好，实验才没有成功。当时我感觉有一点囧，更多的是一种幸福，只有和你比较熟的人才会不顾情面地说你啊。那都因为是你，但是你不知道。还有一个镜头，就是在那个时间里，你在后面的黑板上写写画画，比较一本正经的装逼，俨然是一个老师。我就坐在后面，静静地看着你，听着你说的话，那一刻我希望时间就停留在那，forever。

高中的时候，你和我不在同一个学校，我连见你的机会都没有。有次，在qq上和你说了两句话，我觉得是那么的幸福。我还记得当时你说你和同学在喝糖水，应该是个星期六的晚上，不知道你还记得吗。有一次，我们在药店遇到，当时的每一句话，我没有详细地记住，但是啊，我在我日记本中记录了这一切。那个药店，那一天，每个细节，我都像把它记录下来，因为有你在里面。

高考出成绩的时候，当时考得比我还高，但是你好像还是不是很满意，我还在你的那个说说下面说，你已经是公主了，而你不知道。在我心中，你一直都是那个很认真很努力很有正能量的人，你就是公主。

大学了，qq上你有一次主动找我，是因为你学姐的淘宝店，想让我给你推一下。我照做。2015年新年的时候，你见到我的时候说，我瘦了。探年例的时候，我故意表现得不在意你，还有那时的我和平时的我不一样，那是因为你在啊，我想引起你的注意，就像当年一样。2016年的时候，我想把这一切告诉你的时候，你说，你已经有男朋友了。那一刻，我仿佛一下子变得一无所有，有种心痛的感觉，有种想哭的感觉。但是在微信上我还是说了，希望你和你男朋友来西安玩。

后来，我在微博上看到你过得很开心。后来，我就取消关注你了。

今年再见你，没有想象中尴尬，静静地听着你说话，怕惊动了时光。

最后，祝福你，我曾经喜欢过的你。
