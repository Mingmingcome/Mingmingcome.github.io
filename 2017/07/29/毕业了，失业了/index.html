毕业了，失业了

发表于 2017-07-29   |   分类于 随笔   |     |   阅读次数 
2017年7月29日22:00:34
歌名：陈奕迅《明年今日》
不知道为什么想到了这个题目，但这正是我此时此刻的状态：毕业了，失业了。大概是高中的时候，有一种联通卡附带了超级QQ，超级QQ拥有建可以装很多人的多个群的特权。那时，我就给其中一个群起了这个名字：好吧！毕业，失业。当时我是在想，我从“读书”以后从来没有过第二份人生的“工作”，毕业了，就相当于失业了，浓浓的文艺气息有木有。可。。。。。。我没有想到的是这样的预言居然成真了，我的妈呀。大家好，我是个好人，我是预言家。呜呜，欲哭无泪。 失业的原因有而二：第一，能力不足。导师第一次交任务给我的时候，我没有完成。原谅我这个菜鸟，我不知道怎么做。最重要的是，我还不去问，脸皮薄。因为沟通的问题，第一次任务，不了了之，但是我深深地感觉到导师对我的失望。第二，状态未转变。毕业来了之后，我还是那种在学校学习的状态，就是学习要老师来督促，以为所有事情都会是别人安排好，自己可以做就做，不做就不做。但是这是工作啊，公司给钱我，是让我来工作的，不是吊儿郎当地学习的。我从来没有参加过任何形式的工作，除了帮家里种地之外，就是读书了。 这算是我这个职场新人的收获吧。第一：沟通能力很重要，学着去沟通。不懂就问。不懂就问。不懂就问。重要的事情要说三遍。第二：你已经是社会人，工作就是工作。赶紧醒过来吧，骚年，别再沉迷于学校的记忆了。 失业了之后，心里满是失落，挫败的感觉，以为人生就这样完了。本人某211高校本科毕业学渣，从小到大都是学习、学习、学习，什么技能都没有学会，我不是说那种偏理论的知识，而是面对生活的知识，面对人生的知识。这些可能都是学不来的，只有实践过后才知道。我知道遇到挫折要坚强，要自强不息，要乐观，初中的书本里面就已经写得很清楚，当年背得是那样的辛苦，一点也不理解真谛。以前遇到的挫折，最多就是考差了，然后下次在考回来，反反复复就是这样。当人事部跟我说，接下来就去找工作吧。我当时整个人都蒙了。之前我有过太多关于这个工作的愿望和想法了，有了这个工作，今年我就可以把大学欠的贷款还完，我还能给自己买一份生日礼物。我在2017年7月6日，写下这样的文字： 生于1992年12月6日，农历壬申年【猴年】辛亥月丙辰日（十一月十三日），记得妈妈说过我是十一点多出生的，忘了是早上还是晚上了，应该是子时还是午时呢？星座是射手座。 今年2017年12月30日，实打实的，我已经25岁了。刚才忽然想给自己庆祝一下，买一套卖唱设备，一个好一点的吉他，一个声卡，一个麦克风。 这种人生的“惊喜”也不是每个人都会有的。我哭了，很无助，很绝望。这个世界上，还有我的立足之地吗？天晓得。最重要的是，我还胡思乱想。想着我身边的同学朋友都有工作，有着不错的收入，而我成了无业游民，在这个寸土寸金的城市里，没工作就好像被这个世界遗弃一样。 后来，到了找工作的时候，发现都是社招都是1-3年的经验，我只是刚毕业。更尴尬的是，我还要编好我的离职理由，我是那种很老实很老实的人，面对陌生人不说谎都紧张，说谎更不容易了。现在面了几家，一家创业公司，坏境超好，然而机试没过，其他都是外包、培训。我去到外包、培训公司，我第一次觉得我以前什么牛逼哄哄的，现在已经低到尘土中了。 不知道接下来的路会在哪。但是有一群人在温暖着我，他们就是我的同学朋友们。我就在朋友圈上发了： 毕业了，失业了，尝试去改变自己，和你聊得很happy 然后很多同学就留言或微信语音给我，问我怎么样了。知道事情原委之后，不断鼓励我，给我指明我的不足和出路，还提供巨额的金钱支援，无论在本省还是外省。虽然路还是要自己走，但是有这些朋友，走下去有意思多了。 给自己一个机会，无论发生什么，都知道这个世界上不只是你一个人，还有很多爱你的人、关心的人。相信这个世界的美好。听着《明年今日》，却回不到那年高三电教放给我们听的时候了，人生还得继续，且行且珍惜。
